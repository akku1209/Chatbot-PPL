#CHATBOT
##PPL PROJECT
Chat bot was created using Python.
Libraries like 
1. numpy
2. tensorflow
3. tflearn
4. json and 
5. pickle
were used.

Before running the code, the necessary packages are required to be installed.
Use the following command to do so:
<b> pip install `tensorflow`, `keras`, `pickle`, `nltk` respectively</b>

Run the code using:
<b> python main.py </b>
